- AWS CLI 2.11.16 Python/3.11.3 Darwin/22.4.0

## install redis

- brew install redis
- brew services start redis

## keyhot 快捷鍵

- alt+s 可以新增狀態
- alt+c 可以新增卡片
- alt+d 可以切換到日曆頁面
- alt+m 可以切到聊天室
- alt+w 可以切到任務列表
- alt+f 可以切到成員
