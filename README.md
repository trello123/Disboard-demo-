# Disboard

網站: https://www.disboard.online/
<img width="960" alt="disboard" src="https://i.imgur.com/OllZTBh.png">

當你覺得事情好多好雜，私人通訊軟體不時跳出公事訊息，或總是忘記事情優先順序時，交給 Disboard 幫你處理就對了！<br>
讓項目管理，變得更簡單；讓團隊協作，無往而不利。

## Disboard 是向專案管理網站 Trello 跟通訊軟體 Discord 致敬的專案管理網站。

- **任務列表**：規劃想要完成的任務，並協助進行分類
  <img width="960" alt="任務列表" src="https://i.imgur.com/gS1LG0W.png">

- **日曆功能**：根據每個任務的起始時間，掌握專案進度
  <img width="960" alt="日曆功能" src="https://i.imgur.com/QVGdcgz.png">

- **即時聊天**：同一專案中的每個成員，能夠有效溝通，增進專案開發效率
  <img width="960" alt="即時聊天" src="https://i.imgur.com/WPxDI3t.png">

- **群組功能**：輕鬆組織專案成員，方便管理，協作快速
  <img width="960" alt="群組功能" src="https://i.imgur.com/tiZMMcx.png">

- **圖表顯示**：迅速得知專案開發完成度<br/>
  <img width="960" alt="圖表顯示" src="https://i.imgur.com/3HyLtRX.png">

## 使用技術

- **前端**： HTML / CSS / Tailwind CSS /JavaScript / Stimulus JS / Turbo
- **後端**： Ruby on Rails / Action Cable
- **部署**： Heroku
- **版本控制**： Git / GitHub
- **資料庫**： PostgreSQL / Redis
- **版面設計**： Figma
- **第三方登入**： Google / GitHub
- **金流串接**： 藍新金流

## 使用版本

- Ruby 3.2.1
- Rails 6.1.7.3
- Node v16.19.0
- PostgreSQL 15.2

## install redis

- brew install redis
- brew services start redis

## hotkey 快捷鍵

- option/alt + s 可以新增狀態
- option/alt + c 可以新增卡片
- option/alt + w 可以切換到日曆頁面
- option/alt + e 可以切到聊天室
- option/alt + q 可以切到任務列表
- option/alt + r 可以切到成員
