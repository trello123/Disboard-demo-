# README

## 使用技術

- ruby 3.2.1
- Rails 6.1.7.3
- node v16.19.0
- PostgreSQL 15.2

## install redis

```bash
brew install redis
brew services start redis
```

## hotkey 快捷鍵

- alt + s 新增狀態
- alt + c 新增卡片
- alt + d 切換到日曆頁面
- alt + m 切到聊天室
- alt + w 切到任務列表
- alt + f 切到成員
